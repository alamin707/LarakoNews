# Installation

# Server Requirements

Before you install LarakoNews make sure your server meets the following requirements:

  • PHP >= 7.1.3
  • MySQL 5.6+
  • Intl PHP Extension
  • OpenSSL PHP Extension
  • PDO PHP Extension
  • Mbstring PHP Extension
  • Tokenizer PHP Extension
  • XML PHP Extension
  • Ctype PHP Extension
  • JSON PHP Extension

# Installing LarakoNews

# Step 1: Extract and Upload

Unzip the downloaded archive package. Rename the LarakoNews directory to your desired directory name and upload the directory to your web server through FTP or Control Panel.

# Step 2: Creating Database

Create a database for LarakoNews or choose a database name through your server control panel. If your server has phpMyAdmin, then you can also create a database using phpMyAdmin.

# Step 3: Run Installation Wizard

Go to your website address ( example.com/install ), then you'll see an installation wizard.

WARNING

You should configure your web server's document / web root to be the public directory.
